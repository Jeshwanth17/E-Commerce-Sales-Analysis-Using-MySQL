# E-Commerce Sales Analysis Using MySQL

## Project Overview

This project analyzes e-commerce sales data using MySQL to generate business insights related to customer behavior, product performance, sales trends, and revenue generation.

## Business Problems Solved

1. Top 5 Selling Products by Quantity
2. Most Frequently Cancelled Products
3. Peak Purchase Time Analysis
4. Top Spending Customers
5. Highest Revenue Product Categories
6. Return and Cancellation Rate Analysis
7. Preferred Payment Methods
8. Customer Age Group Purchasing Behavior
9. Monthly Sales Trends
10. Gender-wise Product Category Preferences

## SQL Skills Demonstrated

- Data Aggregation (SUM, COUNT)
- GROUP BY and ORDER BY
- CASE Statements
- Date Functions
- Data Cleaning and Transformation
- Business KPI Analysis

## Key Insights

- Identified the highest-selling products.
- Analyzed customer spending patterns.
- Determined peak shopping hours.
- Measured cancellation and return rates.
- Evaluated revenue contribution by product category.
- Examined demographic purchasing behavior.
